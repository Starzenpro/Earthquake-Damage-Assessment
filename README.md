## 📋 Project Overview

A data analysis project examining building damage patterns from the 2015 Nepal earthquake in Kavrepalanchok district. This analysis helps understand structural vulnerabilities and class balance in damage assessment data.

## 🎯 Key Features

- **Class Balance Analysis**: Visualize damage severity distribution
- **Data Generation**: Synthetic earthquake damage data for testing
- **Interactive Notebooks**: Jupyter notebooks with complete analysis
- **Professional Visualizations**: Publication-quality charts
